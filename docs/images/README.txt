SCREENSHOT PLACEHOLDERS

This directory is for storing screenshots of the Student Management System.

Please add your own screenshots of the application with filenames matching those
referenced in the main README.md file:

- login-screen.png
- registration-screen.png
- admin-dashboard.png
- student-management.png
- course-management.png

You can capture these screenshots by running the application and using your
operating system's screenshot capabilities:

Windows: Press Windows+Shift+S
macOS: Press Command+Shift+4
Linux: Use the Screenshot tool or press Print Screen

Save the images in this directory to make them appear in the documentation.
